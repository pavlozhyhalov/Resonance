# Resonance

Дихання за методом Віма Хофа, практика холоду, музика по частотах (через **YouTube Music**), календар активності з аналізом, завдання й винагороди. Дані синхронізуються через окремий проєкт Supabase.

## Музика
- **Частоти** — тап по треку відкриває його в **застосунку YouTube Music** саме на цій частоті (працює надійно, не буває «відео недоступне»). Збоку кнопка-календар — відмітити прослуховування (1 хв = 1 бал).
- **Дихання / Холод** — кнопка «♪ Увімкнути музику в YouTube Music» відкриває релакс-музику в додатку; повернись у застосунок і запускай практику. Короткі сигнали вдиху/видиху/затримки лишились (їх можна вимкнути перемикачем «Звукові підказки»).
- Жодного синтезованого фону — лише твоя музика з YouTube Music.

## Оновлення (нова версія)
1. Розпакуй `resonance-app.zip`.
2. github.com → репозиторій → **Add file → Upload files** → перетягни **весь вміст** → **Commit changes**.
3. Онови сайт. (Старі файли в `audio/` окрім `cue-*.mp3` більше не потрібні — можеш видалити, але вони не заважають.)

## Вхід
Реєстрація працює **без підтвердження пошти** — створив акаунт і одразу в застосунку. Той самий акаунт на телефоні й компʼютері = спільні дані.

## Розділи
- **Дихання** — раунди, кількість дихань (повзунок), темп, відновлення, підказки. Затримку завершуєш подвійним тапом по екрану або кнопкою. Рання зупинка теж пишеться в календар.
- **Холод** — душ/ванна, тривалість, круговий таймер.
- **Календар** — мітки активності + кнопка «Аналіз» (діаграми: метрика/період/тип).
- **Цілі** — завдання дають бали; 1 хв практики = 1 бал; бали → винагороди.

## Технічно
- Окремий проєкт Supabase `Resonance` (eu-west-1), таблиці з RLS. Ключ у `js/config.js` — публічний.
- Частоти: чесно — наукова база лише в бінаурал-ритмів (попередня); Solfeggio/432 — традиція без доказів.
