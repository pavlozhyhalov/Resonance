# Resonance

Дихальні практики, холод, музика по частотах (через YouTube Music), календар з аналізом, завдання й винагороди. Бекенд — окремий проєкт Supabase.

## Що нового
- **Дихання — це тепер розділ із практиками:** Метод Віма Хофа, Квадратне дихання (4-4-4-4), 4-7-8, Когерентне. Кожна — окрема сторінка з описом, користю й таймером.
- **Частоти:** тап по частоті → детальна сторінка з поясненням (для чого / як впливає) і **5 посиланнями на вибір** у YouTube Music. Вся вкладка більше нікуди не «перекидає».
- **Час рахується правильно:** у диханні зараховується лише час **у затримці**, у холоді — лише час у холоді.
- **Звук вдиху/видиху** замінено на мʼякий «подих» (можна вимкнути).
- **Аналіз:** доданий період **«Увесь час»** (довгі діапазони групуються по тижнях).
- Скрізь додано **інформацію**: для чого практика і що дає тілу/життю.

## Оновлення (нова версія)

## Вхід
Реєстрація **без підтвердження пошти** — створив акаунт і одразу в застосунку. Один акаунт = спільні дані на всіх пристроях.

## Розділи
- **Частоти** → опис + 5 треків у YouTube Music. Кнопка «Відмітити прослуховування» (1 хв = 1 бал).
- **Дихання** → список практик; кожна веде на свою сторінку.
- **Холод** → душ/ванна, таймер, опис користі.
- **Календар** → мітки активності + «Аналіз» (метрика/період/тип, у т.ч. «Увесь час»).
- **Цілі** → бали за корисний час, обмін на винагороди.

## Чесно
Наукова база є лише в бінаурал-ритмів (попередня). Solfeggio/432 Гц — традиція без доказів (позначено «Без доказів»); цінність — у спокої та усвідомленому слуханні.
